EduCalc Predictor

JAVA ICSE LOGIC ALGORITHM

This project is a Java-based utility designed specifically for Class 10 ICSE students. It implements the "Best of Five" rule used by the Board to calculate percentages, including the compulsory English paper logic.

EduCalc.java 
import java.util.*;

// ICSE Percentage Predictor Logic
public class EduCalc {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Compulsory Subject: English
        System.out.print("Enter English Marks: ");
        int eng = sc.nextInt();

        int[] otherSubjects = new int[5];
        // Input other 5 subjects
        for(int i=0; i < 5; i++) {
            System.out.print("Enter Subject " + (i+1) + " marks: ");
            otherSubjects[i] = sc.nextInt();
        }

        Arrays.sort(otherSubjects);

        // Best of 5 Logic: English + Top 4 others
        int total = eng;
        for(int i=4; i >= 1; i--) {
            total += otherSubjects[i];
        }

        double percentage = total / 5.0;
        System.out.println("Your ICSE Percentage is: " + percentage + "%");
    }
}
