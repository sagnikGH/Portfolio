SortViz

JAVAALGORITHMSVISUALIZATION

A tool to visualize how sorting algorithms work. This uses the standard Bubble Sort algorithm taught in the ICSE curriculum but adds a graphical layer.

BubbleSort.java
public void bubbleSort(int[] arr) {
    int n = arr.length;
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            // ICSE logic: compare adjacent elements
            if (arr[j] > arr[j + 1]) {
                // Swap elements
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                // In SortViz, we trigger a UI update here
                updateVisuals();
            }
        }
    }
}
